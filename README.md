## Built with
* [TypeScript](https://typescriptlang.org)
* [Angular 2](https://angular.io)
* [Ionic 2](http://ionicframework.com)
* [Cordova](https://cordova.apache.org)

## Getting started

```bash
git clone https://github.com/manduro/onontdekt.git
cd onontdekt
npm install
npm start
```

### Run unit tests
```bash
npm test
```

### Debug unit tests
```bash
npm run karma
```

### Run E2E
```bash
npm start
npm run e2e
```
